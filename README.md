# javascript-lunch

우아한테크코스 레벨1 점심 뭐 먹지 미션

### [🍕배포페이지🍔](https://leejin-yang.github.io/javascript-lunch/)

## 실행 방법

1. 레포지토리 클론

```bash
git clone -b leejin-yang --single-branch https://github.com/woowacourse/javascript-lunch.git
```

2. 의존성 설치

```bash
npm install
```

3. 실행

```bash
npm run start
```
